# wckmp

*Project for research purposes.*

**wc** - print the number of lines, ~~words, and bytes~~ in files.

Based on Kotlin Multiplatform.

## How to use

For posix platforms:

```
./posix-app.kexe file1.txt /home/user/myjsonfile.json
```

Output:

```
file1.txt : 51
/home/user/myjsonfile.json : 2412
```
